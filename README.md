  Within SimuSin.mat, the Raw.mat contains the raw data for the sin simulation signal. The SNR.mat shows the specific signal-to-noise ratio of the sin signal, the CC.mat represents the raw data used to find the boundary order using the correlation coefficient method, the PE.mat represents the raw data used to find the boundary order using the Permutation Entropy method, and the HD.mat represents the raw data used to find the boundary order using the Hausdorff Distance method.
The structure of SimuBumps.mat is the same.
ThresholdFunction.mat contains the raw data from comparative experiments using five different threshold functions.
EEMD_SimuBumps.mat contains the raw data from all imfs obtained by decomposing the Bumps signal using EEMD.
ExperimentUwbPos.mat contains all the raw data from the UWB experiments.
